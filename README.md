# MiMo-Calculus-Tutor-Agent
使用MiMo模型开发的个性化微积分学习Agent，帮助自学者通过交互式引导提升数学思维

## 项目背景

我目前主要使用ChatGPT辅助自学大学微积分。在学习复合函数积分求导等章节时，我会把自己的解题过程发给AI，让AI进行详细纠错、思维点评和正确求导演示。AI不仅指出错误，还会给出结构化总结和考点记忆，显著提高了我的学习效率。

## Agent开发目标

计划开发一个真正的**AI数学导师Agent**，核心功能包括：

- 实时评估学生当前微积分掌握水平
- 自动生成个性化练习题和学习路径
- **交互式逐步引导解题**（只给提示和纠错，不直接给完整答案）
- 结构化知识输出（公式推导 + 问题总结 + 一句话考点记忆）
- 集成SymPy进行精确符号计算验证
- 长期记忆，追踪薄弱知识点

## 当前进度

- 已积累多轮高质量ChatGPT微积分学习对话记录（纠错 + 正确求导 + 总结）
- 使用Prompt工程模拟Agent工作流（Chain of Thought + 纠错循环）
- 准备接入 **小米MiMo模型** 进行实际开发和测试
