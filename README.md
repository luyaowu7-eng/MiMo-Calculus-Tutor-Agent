# MiMo-Calculus-Tutor-Agent

使用 **MiMo模型** 开发的个性化微积分学习Agent，帮助自学者通过交互式引导提升数学思维和解题能力。

## 项目背景

我目前主要使用 **ChatGPT** 辅助自学大学微积分。在学习**复合函数积分求导**等章节时，我会把自己的解题过程发给AI，让AI进行**详细纠错、思维点评和正确求导演示**。AI不仅精准指出了我解题中的错误（上下限处理、求导遗漏等），还会给出结构化总结和“一句话记忆考点”，显著提高了我的学习效率和数学思维能力。

## Agent开发目标

计划开发一个真正的 **AI数学导师Agent**，核心功能包括：

- 实时评估学生当前微积分掌握水平
- 自动生成个性化练习题和学习路径
- **交互式逐步引导解题**（只给提示、纠错，不直接给完整答案，培养独立思考）
- 结构化知识输出（公式推导 + 问题总结 + 一句话考点记忆）
- 集成 **SymPy** 进行精确符号计算验证
- 长期记忆，动态追踪薄弱知识点并调整难度

## 当前进度
