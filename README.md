# MiMo-Calculus-Tutor-Agent
使用MiMo模型开发的个性化微积分学习Agent，帮助自学者通过交互式引导提升数学思维
## 项目目标
帮助自学者通过**交互式逐步引导**的方式掌握大学微积分（极限、导数、积分、多变量等），重点培养数学思维而不是死记硬背。

## 当前成果
- 已使用 **ChatGPT** 进行多轮深度学习对话
- AI帮助完成复合函数积分求导等题目的**纠错、正确求导过程演示、问题总结和考点记忆**
- 积累了高质量学习对话记录和反思笔记（见 screenshots 文件夹）

## 未来计划（使用MiMo开发）
- 实时评估学生掌握水平
- 自动生成个性化练习题和学习路径
- 支持**交互式解题**（输入步骤 → Agent给出提示/纠错）
- 集成SymPy进行符号计算验证
- 结构化输出：公式推导 + 一句话考点记忆

欢迎使用MiMo Token权益来加速这个Agent的开发！
